## ✅ วิธีแก้ (เลือกอย่างใดอย่างหนึ่ง)

### วิธีที่ 1: สร้างห้องใหม่ (แนะนำ) ⭐

1. กด **Hard Refresh**: `Cmd + Shift + R` (Mac) หรือ `Ctrl + Shift + R` (Windows)
2. ไปที่หน้าแรก: http://localhost:3000
3. คลิก **"สร้างห้อง"**
4. เชิญเพื่อนเข้ามาใหม่
5. **เริ่มเกมใหม่**

### วิธีที่ 2: Clear Browser Cache

1. เปิด Chrome DevTools (F12)
2. ไปที่ tab **Application**
3. คลิก **Clear storage** ทางซ้าย
4. คลิก **Clear site data**
5. Refresh หน้าเว็บ

### วิธีที่ 3: ใช้ Incognito Mode

1. เปิด Chrome Incognito Window: `Cmd + Shift + N` (Mac) หรือ `Ctrl + Shift + N` (Windows)
2. ไปที่ http://localhost:3000
3. สร้างห้องใหม่และเล่น
